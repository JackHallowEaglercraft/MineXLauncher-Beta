# 🎮 Eaglercraft

> A web-based Minecraft client that runs directly in your browser—no installation or authentication required!

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![GitHub](https://img.shields.io/badge/GitHub-Eaglercraft-brightgreen)](https://github.com)
[![Browser Support](https://img.shields.io/badge/Browser-WebGL_Required-orange)](https://caniuse.com/webgl)

## 🌟 Features

- 🌐 **Play in Your Browser** - No installation, no launcher, no Minecraft account required
- 🎮 **Classic Minecraft Experience** - Based on Minecraft Beta 1.3 and 1.5.2
- 🏗️ **Singleplayer Worlds** - Create and explore your own sandbox worlds
- 👥 **Multiplayer Support** - Connect to servers and play with friends
- ⚡ **Lightweight** - Optimized performance for modern browsers
- 🔧 **Customizable** - Easy to modify and extend
- 💾 **Save Progress** - Your worlds persist in browser storage

## 🚀 Quick Start

### Requirements
- Modern web browser with WebGL support (Chrome, Firefox, Edge, Safari)
- JavaScript enabled
- At least 512MB RAM recommended

### Getting Started

1. **Visit the Website**
   ```
   Open your browser and navigate to the Eaglercraft server
   ```

2. **Create a World**
   - Click "Singleplayer"
   - Enter a world name
   - Choose your settings
   - Start playing!

3. **Multiplayer**
   - Click "Multiplayer"
   - Enter server address
   - Join and play with friends

## 🛠️ Technologies Used

- **JavaScript (ES6+)** - Core game logic
- **WebGL** - 3D graphics rendering
- **HTML5 Canvas** - Rendering and UI
- **WebSockets** - Multiplayer connectivity
- **IndexedDB** - Client-side world storage
